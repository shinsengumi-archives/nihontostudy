Good morning to all,

Q1: how many hours did this project take?
Arround 73 hours.

Q2: what tools did you use?
A jewerly saw 0.04 (you can see it in the pics), a electric drill, and a 200
gr. hammer, (also I use a file to make some marks inside the part where goes
the cooper )

Q3: what process to get the final aged look? (if some one was for asking
this)
In this special case (because it has a cooper insert) I can not use any
liquid to fast oxidize, becuase the cooper contaminate the steel with a red
colour (as the silver in black).
I can use acid vapor but it is too toxic, and you must to control the
process every 5 min for 7 to 12 hours).
So the only way that I find, was with salt, water, heat and a lot of
patience
