NihontoAuctions Update

by deron_vfr750

Greetings,
I've been working with Rich Turner most of the weekend on the Reserve trigger. And it looks like we have the problem fixed. This week, after the last auction ends, I will place the NihontoAuctions website into test mode and allow our testers to go over the system one final time. While they are doing this I will be setting up the new server and domain name.

All auctions and bids as well as user names and passwords will be transferred to the new server.

Once the testers give me the green light, I will open www.NihontoAuctions.com up for business.

As of this date, I'm happy with all the changes we've made and feel comfortable that everything is working as planned. When I first introduced NihontoAuctions two weeks ago we didn't have all the new things we have now. The system was stable but not as robust as we would have liked. With the creation of more features (as a direct result from the Nihonto Community), we introduced some additional problems, that have since been fixed.

I think we're set to go!

I will post again when the new site is up and running and would like to invite everyone to join us at www.NihontoAuctions.com.

Thank you for the band-width.

Cheers,
