More questions today...

First question. I've never picked up on this before, but realized that both NAMBAN and NANBAN is used with describing these tsuba. Is there a simple explanation for this?

Second question. What is the difference between the carving techniques, SUKIDASHI-BORI and SUKISAGE-BORI? My translated texts use the term sukisage-bori but glossaries seem to only define sukidashi-bori. I'm suspecting they describe the same technique of producing low relief by carving away the tsuba's surface metal.

Just more confusion discovered when writing was was to be a simple, straight-forward namban tsuba article. (sigh)

Re: [nihontostudy] Re: Name this dragon...

by robsongord

Hi Ron,

It is, as Bob says, a pronunciation/spelling thing. The actual
pronunciation is /m/, so Namban. It is very difficult to pronounce /n/
after the initial /n/, but some sources use that as the spelling. It
is similar to the nengo Tenbun. Some sources give Tembun. I always use
Namban and Tenbun, but some other sources use the alternative spelling.

The carving explanation sounds reasonable, but I think there is a fair
degree of ambiguity over many of these terms. And, thanks for getting
back on the kanji stuff.

Gordon
