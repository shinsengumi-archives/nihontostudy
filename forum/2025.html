Hamon description

by sworddad

Curious about the correct way to describe a particular style of hamon which I'd describe as a smallish gunome with ko-ashi, in groups of five or six gunome. Each section of gunome is seperated with a slightly larger ashi, which while subtle, does make the straight gunmome hamon appear in "groups" of gunome. Hope that is clear.

One term I find is called "sugu ashi iri" which I take to mean, "suguha containing ashi."

The pattern I'm trying to describe would see (to me) correctly described as something like "sugu kaku (boxed) gunome ashi iri" or am I sounding too midwestern with my Japanese?? Any comments? If a small sketch (file) would help, I'll send one out.

Ron
