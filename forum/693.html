Alan B.- Milt's Hikone bori tsuba mei.

by ccikkyudo

Alan,
If you have the Kinko Jiten handy (Heisei 11 ed.)
check the Mokuji (table of contents) page 42, bottom
for Milt's first character.
Sakusha recorded from page 209.

Reading given is Yoshi or Nobu (kunyomi) or Gi (onyomi).
This is new Nelsons #1317.
Eight strokes.

This is not to be confused with the Sen (onyomi)
Nobu (kunyomi) character on page 29, bottom.
This is new Nelson's #1328.
Nine strokes.
Sakusha recorded from page 920.
