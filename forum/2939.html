Yari type

by sworddad

Hi All -

Can anyone tell me the correct (accepted today) name for a yari I'm making a record sheet on today? To describe it, it is a stubby, short (4") head, 4-sided yari made with yokote. This one is quite nice and signed by Yokoyama Bizen Sukehira (shinshin-to).

In Kajihara's glossery (p.250) illustrates what I would say is that shape/form, but a number of names are given for it: tanpo yari, te yari, ko-yari, kago yari, mijika-yari.

The mijika-yari elaborates, saying "short, straight yari of 3 to 6 sun." This sounds like what my yari would be but then the other types are simply identified by a name (which I can not translate, other than the ko-yari). Any answer to this?

Can anyone comment on Knutsen's latest publication on pole arms, if it is useful...cost and sources? I've yet to buy a copy and have already forgotten the name of this latest Knutsen publication.

Ron

Re: [nihontostudy] Yari type

by davidzincavage

There aren't a lot of other choices in English besides the older, very scarce and expensive earlier title by the same author. It's 40 pounds from:

http://www.globaloriental.co.uk/book.asp?Title_ID=13

A blade without projections is a su yari. A four-sided spear blade is a ryo-shinogi-yari.
